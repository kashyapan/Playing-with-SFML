# Playing-with-SFMV

Programs on SFMV from absolute scratch.
Contributions and Suggestions welcomed

## Running app

`g++ -c main.cpp`

`g++ main.o -o sfml-app -lsfml-graphics -lsfml-window -lsfml-system`

`./sfml-app`

### Alternatively use sfml.sh file

`chmod +x sfml.sh`

`. sfml.sh`

`sfml ` < filename >

To run `./sfml-app`
